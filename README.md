# Healthcare Dashboard (B2B SaaS UI)

A scalable frontend application built with React + TypeScript simulating a healthcare analytics and patient management platform.

## 🔧 Tech Stack

* React + TypeScript (Vite)
* Firebase (Auth + Firestore)
* Zustand (State Management)
* Tailwind CSS
* Recharts
* Service Worker (Notifications)

## ✨ Features

* 🔐 Firebase Authentication (Email/Password)
* 📊 Analytics Dashboard with live data
* 👨‍⚕️ Patient Management (CRUD with Firestore)
* 🔄 Grid / List toggle for patient view
* 🔔 Browser Notifications (Service Worker)
* ⚡ Optimistic UI updates + skeleton loaders
* 🧠 Clean state management using Zustand
